# 华南理工大学 22网络工程 梁燊伟 202230441466

## 项目简介
本项目总共设计了 6 个页面，分别是：
1. 主页
2. 用户注册页
3. 用户登录页
4. 管理员登录页
5. 用户购物车页
6. 用户历史订单页

## 主要功能
项目中主要设计的功能有：
1. **用户购物车功能**，位于 `paycart.js`
2. **用户历史订单功能**，位于 `order.js`
3. **管理员修改商品、更新用户余额、发货功能**，位于 `manager.js`
4. **用户登录、注册、退出登录**，管理员的登录以及商品展示

## 数据库
数据库使用的是 **SQL Server**。
